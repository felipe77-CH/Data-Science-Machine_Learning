Finalmente, se introducirá el comando msno.dendrogram(), que traza un dendrograma (un diagrama de árbol) que mostrará de forma más completa la correlación entre las variables, revelando conexiones más complejas que las visibles en el mapa de calor.

Para construir el dendrograma, el modelo utiliza un algoritmo de agrupamiento jerárquico para agrupar las variables por su correlación nula (si desea comprender mejor cómo se realiza el algoritmo, simplemente eche un vistazo a este enlace). Su interpretación debe hacerse de arriba a abajo.

En la parte inicial igual a 0, que una variable esté vinculada a otra significa que siempre se puede llenar una de ellas cuando la otra está vacía, o ambas siempre se pueden llenar/vaciar.

En divisiones cercanas a 0, los atributos se predicen bien entre sí, pero no perfectamente como lo hicieron en la fila 0 y así sucesivamente.

Eliminacion de datos perdidos¶

El código df1 = df.dropna() elimina todas las filas del DataFrame df que contienen al menos un valor nulo. La función dropna() devuelve un nuevo DataFrame donde se han eliminado todas las filas que contenían valores nulos en alguna de sus columnas. En otras palabras, df1 contendrá solo las filas que no tienen ningún valor nulo en ninguna de sus columnas.

Una vez que tenga una herramienta para predecir datos, podrá usarla para predecir datos faltantes. La tarea general de definir valores para Los valores faltantes se llaman imputación. Si está imputando datos, necesitará crear una canalización y utilizar la misma lógica de imputación durante la creación del modelo y tiempo de predicción. La clase SimpleImputer en scikit-learn Manejar valores de características medias, medianas y más frecuentes. El comportamiento predeterminado es calcular la media:

Proporcione estrategia='mediana' o estrategia='más_frecuente' a cambie el valor reemplazado a mediana o más común, respectivamente. Si desea completar con un valor constante, digamos -1, use estrategia='constante' en combinación con fill_value=-1.

CONSEJO Puedes usar el método fillna en pandas para imputar valores faltantes también. Asegúrate de no filtrar datos aunque. Si está completando con el valor medio, asegúrese de utiliza el mismo valor medio durante la creación del modelo y tiempo de predicción del modelo.

Las estrategias más frecuentes y constantes se pueden utilizar con datos numéricos o de cadena. La media y la mediana requieren valores numéricos. fecha. La biblioteca fancyimpute implementa muchos algoritmos y sigue la interfaz scikit-learn. Lamentablemente, la mayoría de los algoritmos son transductivo, lo que significa que no se puede llamar al .transform método por sí solo después de ajustar el algoritmo. El iterativo soy la computadora es inductiva (desde entonces ha sido migrada de fancyimpute to scikit-learn) y admite la transformación después del ajuste.

def add_indicator(col):
    def wrapper(df):
        return df[col].isna().astype(int)
    return wrapper
df1 = df.assign(cabin_missing=add_indicator("cabin"))
print(df1)
