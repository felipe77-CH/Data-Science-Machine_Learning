Una vez que tenga una herramienta para predecir datos, podr usarla para predecir datos faltantes. La tarea general de definir valores para Los valores faltantes se llaman imputacin. Si est imputando datos, necesitar crear una canalizacin y utilizar la misma lgica de imputacin durante la creacin del modelo y tiempo de prediccin. La clase SimpleImputer en scikit-learn Manejar valores de caractersticas medias, medianas y ms frecuentes. El comportamiento predeterminado es calcular la media:

Proporcione estrategia='mediana' o estrategia='ms_frecuente' a cambie el valor reemplazado a mediana o ms comn, respectivamente. Si desea completar con un valor constante, digamos -1, use estrategia='constante' en combinacin con fill_value=-1.

CONSEJO Puedes usar el mtodo fillna en pandas para imputar valores faltantes tambin. Asegrate de no filtrar datos aunque. Si est completando con el valor medio, asegrese de utiliza el mismo valor medio durante la creacin del modelo y tiempo de prediccin del modelo.

Las estrategias ms frecuentes y constantes se pueden utilizar con datos numricos o de cadena. La media y la mediana requieren valores numricos. fecha. La biblioteca fancyimpute implementa muchos algoritmos y sigue la interfaz scikit-learn. Lamentablemente, la mayora de los algoritmos son transductivo, lo que significa que no se puede llamar al .transform mtodo por s solo despus de ajustar el algoritmo. El iterativo soy la computadora es inductiva (desde entonces ha sido migrada de fancyimpute to scikit-learn) y admite la transformacin despus del ajuste.

def add_indicator(col):
    def wrapper(df):
        return df[col].isna().astype(int)
    return wrapper
df1 = df.assign(cabin_missing=add_indicator("cabin"))
